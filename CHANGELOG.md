# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.0.1] - 2025-01-18

### Added
- Initial release of DiffLens
- Visual regression testing with Playwright
- Accessibility auditing with axe-core
- CLI tool with `init` and `test` commands
- MCP server for AI agent integration
- REST API server for programmatic access
- HTML report generation
- Support for multiple viewports per scenario
- Action support: click, type, wait
- Mask and hide selectors for dynamic content
- JSON and AI-friendly output formats
- TypeScript configuration file support
